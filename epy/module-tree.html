Module Hierarchy

  • src.CharCreator: CharCreator, will house the character generator / creator allowing players to create custom chars
  • src.ColorPicker: This is a helper class to create a color picker for the players pathlines
  • src.CombatLog: Reworked textArea to be a read only combat log from pgu
  • src.Cursors: Various cursors used by the game
  • src.Game: Main file for the project till I have time to refactor the code to something more managable
  • src.Job: This will be the main job class, that the other jobs will fill out as a template
  • src.JobList: This will create a list of all the different jobs based on the config files in the jobs directory
  • src.MapTile: Basic MapTile container for tile data
  • src.MonsterGenerator: This will allow adding monster dynamically from a set of configuration files in the 'mobs' directory, images for the monsters should be put in the images directory.
  • src.Player
  • src.Room
  • src.Stats: This will be for the statistics window
